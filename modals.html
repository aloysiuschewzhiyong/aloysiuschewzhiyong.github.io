Speed typing game

Web Application

HTML

CSS

Javascript


Description

  • "Kachow" is a web based speed typing game built using HTML, JavaScript and CSS. A fun way to enhance your typing skills. You can challenge yourself to type as accurately and quickly as possible, all while enjoying a visually appealing and responsive user interface.

  • This small side project was fueled by my personal passion for typing on keyboards. One of my hobbies are to build and type on custom keyboards which was what gave me the idea to try and develop a typing practice game like this. This was also done to further practice my skills with HTML, CSS and JavaScript.

  • Built with HTML, CSS, and JavaScript, the design is clean and responsive, ensuring a seamless user experience across various devices.

How to Use:

  1. Click on the input field or press any key to start typing

  2. As you type, characters will be highlighted based on correctness (green for correct, red for incorrect).

  3. Keep an eye on the countdown timer and try to complete the paragraph before time runs out

  4. Your real-time statistics, including WPM, CPM, and mistake count, are displayed below

  5. If you wish to try a different paragraph or reset the game, click the "Try Again" button
