Link Shrinkernator

Web Application

HTML

CSS

Javascript


Description

  • The Link Shrinkernator is a simple web application that allows users to shorten long URLs into concise and shareable links.

  • This small side project that was done as a way for me to familiarize myself with working integrating APIs into web apps with Javascript (Understanding how to structure the API request, handle the response, and update the UI accordingly)

  • Built with HTML, CSS, and JavaScript, the design is clean and responsive, ensuring a seamless user experience across various devices.

How to Use:

  1. Enter the URL to be shortened in the input field

  2. Click the "Submit" button to generate a shortened link.

  3. The shortened link will be displayed below the "Submit" button.

Movie Review Website

Web Application

HTML

CSS

PHP

MySQL


Description

  • CinephilePicks (CP) is a dynamic movie review website. It is made using HTML,CSS and PHP and connects to a MySQL databse

  • This is a project done for an assignment for a Web Application Development module. The purpose of the assignment was to understand and apply the knowledge of website development to develop a working web application in PHP that stores and retrieves data to/from a database server.

  • The application features a well-designed user interface, incorporating PHP for server-side scripting and database interactions, HTML for structuring pages and CSS for styling.

  • It seamlessly integrates with a database server, utilizing SQL queries for CRUD operations. The application persists user data in a defined schema, employing PHP for form handling and client-side interactions

Features

  1. User Login and Registration

  2. Account deletion or deactivation

  3. Password hashing for increased security

  4. Display list movies along with its information through retrieval from database

  5. Search, sort and filter for list of movies

  6. Write/edit/delete movie reviews
